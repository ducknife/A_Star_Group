# A* SQUAD — Frontend

Landing page + trang quản trị của A* SQUAD. React 19 + TypeScript + Vite + Tailwind CSS v4.

Xem tài liệu tổng quan (kiến trúc, cách chạy toàn bộ hệ thống, danh sách trang) tại [README ở thư mục gốc](../README.md).

## Lệnh thường dùng

```bash
npm install       # cài dependency
npm run dev       # chạy dev server tại http://localhost:5173
npm run build     # build production (tsc -b && vite build)
npm run lint      # eslint
```

Cấu hình URL backend qua biến môi trường `VITE_API_BASE_URL` (xem `.env.example`).
